# 17つのデータの型をもとにした分析手法の探索
本書では取り扱うデータによって分析手法を決める決め手としています。

## 【1】x系列のデータ
* 度数分布やヒストグラム
* 母平均の区間推定
* 母平均の検定
* クラスター分析

## 【2】２つのグループのデータ
* 2群の母平均の差の検定
* ウィルコクスンの順位和検定
* 判別分析
* コルモゴロフ・スミルノフの検定

## 【3】３つ以上のグループのデータ
* 対応のある2群の2つの母平均の検定
* ウィルコクスンの符号付き順位検定
